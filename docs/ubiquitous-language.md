# Ubiquitous Language — Billetterie Cinéma (v1)

## Glossaire métier (≥20 termes)

| Terme | Définition | Exemple |
|-------|------------|---------|
| **Séance** | Projection d'un film à une date et heure précises dans une salle donnée | "Séance de 20h15, Avatar 2, Salle 3" |
| **Place** | Siège numéroté dans une salle de cinéma | "Place F12, Rang F, Siège 12" |
| **Réservation** | Engagement temporaire ou définitif sur des places pour une séance | "Réservation #45678 pour 3 places" |
| **Billet** | Titre d'entrée validant l'accès à une séance pour une place donnée | "Billet e-ticket avec QR code" |
| **Salle** | Espace de projection avec capacité et configuration de sièges | "Salle 1, 200 places, écran IMAX" |
| **Paiement** | Transaction financière validant l'achat des billets | "Paiement CB de 32,50€ accepté" |
| **Film** | Œuvre cinématographique projetée | "Film : Dune 2, durée 166min" |
| **Tarif** | Prix d'une place selon catégorie | "Tarif plein : 11,50€, réduit : 8€" |
| **Client** | Utilisateur final qui réserve et achète des places | "Client membre #123456" |
| **Caissier** | Employé gérant les ventes et validations sur place | "Caissier poste 2" |
| **Gestionnaire** | Administrateur du système et de la programmation | "Gestionnaire cinéma multi-salles" |
| **Programmation** | Ensemble des séances planifiées sur une période | "Programmation de la semaine 12" |
| **Catalogue** | Liste des films disponibles à la projection | "Catalogue avec 15 films actifs" |
| **Annulation** | Action de libérer une réservation et ses places | "Annulation avec remboursement à -2h" |
| **Disponibilité** | État d'une place (libre/réservée/occupée) | "43 places disponibles sur 150" |
| **Configuration** | Disposition des sièges dans une salle | "Configuration 12 rangs × 18 sièges" |
| **QR Code** | Code-barres 2D présent sur le billet électronique | "QR Code #AZ123XY" |
| **Remboursement** | Restitution du montant en cas d'annulation valide | "Remboursement de 23€ traité" |
| **Validation** | Contrôle du billet à l'entrée de la salle | "Validation à 19h58 par scan QR" |
| **Notification** | Message automatique envoyé au client | "Email de confirmation envoyé" |
| **Statistiques** | Données de fréquentation et ventes | "Taux de remplissage : 78%" |
| **Abonné** | Client avec carte d'abonnement bénéficiant de tarifs réduits | "Abonné Pass Illimité" |
| **Plein tarif** | Prix standard sans réduction | "11,50€ plein tarif" |
| **Tarif réduit** | Prix avec réduction (étudiant, senior, etc.) | "8€ tarif étudiant" |

## Notes d'usage

- Les termes sont utilisés de manière cohérente dans tout le code et la documentation
- Le vocabulaire métier doit être aligné avec les noms de classes/entités
- Version 2 du glossaire sera enrichie par contexte (Partie 2)
